# Polymarket Quantitative Bot

## Overview
Experimental quantitative trading bot for Polymarket using Frank-Wolfe optimization and Kelly Criterion.

## Project Structure

### Core Modules
- **engine.py**: Frank-Wolfe & Kelly optimization logic
- **validator.py**: Marginal Polytope Constraints validation

### Utilities
- **logger.py**: Performance metrics and loss tracking

### Configuration
- **config.py**: Alpha, Epsilon, and Kelly Fraction parameters

## Entry Point
- **main.py**: Main execution loop
